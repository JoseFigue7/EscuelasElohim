# Imágenes

## Logo de Ministerios Ebenezer

Por favor, coloca el logo de Ministerios Ebenezer en esta carpeta con el nombre:

**`ebenezer-logo.png`**

### Especificaciones recomendadas:
- Formato: PNG (con fondo transparente preferiblemente)
- Tamaño: Mínimo 200px de ancho
- Resolución: Alta calidad (300 DPI o más)
- Fondo: Transparente o blanco

### Ubicación del archivo:
```
frontend/public/images/ebenezer-logo.png
```

El logo se mostrará automáticamente en:
- El header de la aplicación
- La página de login

Si el logo no está disponible, se mostrará el texto "Iglesia de Cristo Elohim" como respaldo.
